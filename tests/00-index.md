# Индекс тестов — Стратегатор

## Назначение
Тест-кейсы для проверки capability на фиксированных входах.

## Кейсы
- TC-001 — Приоритеты и рабочие продукты стратегирования (ST.CAP.04)
- TC-002 — Планирование: перечень задач, план работ, бюджет времени (ST.CAP.07/08)
- TC-003 — Планирование: календарные слоты, распорядок дня, крошки времени (ST.CAP.09/10)
