# Стратегатор — Фаза планирования (методы 07–10)

## Шаг 5: Формулировки работ (метод 07)

**Цель:** перевести задачи в конкретные формулировки работ.

Шаги:
1. Взять список задач по приоритетным проектам
2. Для каждой задачи — сформулировать конкретную работу
3. Указать: роль/исполнитель, день/время, длительность (помидоры или часы), ресурсы
4. Определить ожидаемый рабочий продукт для каждой формулировки
5. Классифицировать: важные/текущие, постоянные/временные
6. Записать формулировки в трекер

**Обязательный стандарт формулировки работы:**
Каждая формулировка ДОЛЖНА содержать:
- Роль исполнителя ("я как [роль]")
- Оценку времени ("2 часа", "3 помидора", "0.5ч")
- Ресурсы (минимум: ноутбук, тихое место, интернет)
- Конкретный рабочий продукт (не "поработать над X", а "написать список Y", "реализовать модуль Z")

Пример хорошей формулировки:
"Я как разработчик в понедельник утром выделю 2 часа (ноутбук + тихое место) на написание технического задания для модуля авторизации."

Если задача расплывчата: уточнить детали или разбить на подзадачи.
Если ресурсы неизвестны: зафиксировать как ограничение, оценить консервативно.

## Шаг 6: Бюджет времени (метод 08)

**Цель:** построить недельный план работ и бюджет времени.

Шаги:
1. Собрать все формулировки работ на неделю
2. Классифицировать: важные/текущие, постоянные/временные
3. Присвоить оценку времени каждой (в помидорах или часах)
4. Суммировать общий бюджет
5. Сравнить с целевым (~30ч): скорректировать если нужно
6. Упорядочить по приоритету и дню
7. Записать план в трекер

Правила:
- Целевой бюджет: ~30 часов в неделю (±1ч)
- Если превышает: предложить деприоритизировать или перенести
- Если меньше: проверить, не упущены ли формулировки
- Включить буфер для срочных работ

## Шаг 7: Расписание и крошки времени (метод 09)

**Цель:** составить расписание дня и список крошек времени.

Шаги:
1. Выделить работы, требующие глубокого фокуса
2. Назначить защищённые слоты в календаре для них
3. Обновить шаблон распорядка дня (пробуждение, еда, спорт, рабочие блоки)
4. Распределить оставшиеся работы по доступным слотам
5. Собрать мелкие задачи в список крошек времени (5-15 мин каждая)
6. Записать расписание и список крошек

Правила:
- Сначала защитить слоты для важных работ
- Расписание = шаблон (повторяется еженедельно) + переменные позиции
- Крошки времени = задачи без глубокого фокуса, выполнимые за 5-15 минут
- Расписание должно быть реалистичным (включая отдых и переходы)
- Если нет свободных слотов: явно предупредить о перегрузке

## Шаг 8: Запись результатов сессии (метод 10)

**Цель:** зафиксировать рабочие продукты сессии.

Шаги:
1. Перечислить все артефакты, созданные или обновлённые за сессию
2. Для каждого: что изменилось и где хранится
3. Записать метаданные сессии: дата, длительность (помидоры), покрытые шаги
4. Отметить отложенные позиции для следующей сессии
5. Сохранить запись сессии

Формат записи:
- Кратко — фокус на что произведено, не на процесс
- Единый формат для трассировки между сессиями
- Если сессия была прервана: записать что завершено, указать причину
- Если новых продуктов нет: "сессия обслуживания — только обзор"
